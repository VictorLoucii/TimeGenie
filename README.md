TimeGenie: AI Timetable Optimizer (Smart India Hackathon Project)

![TimeGenie Demo](./TimeGenieDemo.gif)

TimeGenie is an AI-powered timetable optimization system built for Smart India Hackathon (SIH). It automates the generation of conflict-free academic timetables by modeling complex constraints such as faculty availability, room capacity, NEP-2020 credit structures, electives, and program-wise scheduling rules.
